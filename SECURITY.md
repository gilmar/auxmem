# Security

## Reporting a vulnerability

Please report security issues **privately** by email to [gilmar.souza@gmail.com](mailto:gilmar.souza@gmail.com). Do not open a public GitHub issue for exploitable vulnerabilities.

Include enough detail to reproduce the problem (affected version, steps, impact). You should receive an acknowledgement; response times follow the honest expectations in [CONTRIBUTING.md](CONTRIBUTING.md).

When the project website is live, coordinated disclosure will also be listed at `/.well-known/security.txt` on that site (strategy months 0–1). The repository-side contact policy is this file.
